
#HNB Exchange Median Rate Chart

[DEMO](http://hnbex-median-rates-chart.herokuapp.com/)

API and Libraries:
- [HNB Exchange Rate Lookup API](http://hnbex.eu/api/v1/)
- [jQuery](http://api.jquery.com/)
- [Chart.js](http://www.chartjs.org/docs)
